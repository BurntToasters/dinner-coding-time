Converting Decimal to Binary

The Easy Way

Java gives us a method to convert binary! Of course, teachers usually don't mean for you to use it. They want you to code something yourself. However, methods like this are usually more efficient and bug-tested than anything that a coder could make.

You should always feel justified in using a language's features instead of coding your own: that's what they're there for! To use the Integer.toBinaryString() method, pass any integer to it.

The Boring Way

Some teachers don't like this. As in, get-a-zero-if-you-use-those-methods. Here is some code for binary conversion that uses none of those methods!

It's easiest to understand this when you think of how decimal numbers work. Each digit place is 10 times larger (tens; hundreds; thousands; et cetera). With binary, each digit place is only two times larger, so dividing by 2 "moves" the decimal place 1 smaller. Since this code deals with integers, we don't have to worry about numbers past the decimal point.

From there, the code uses the modulus operator to add the remainder onto the start of the binary string. When it's completed, the result variable will hold a converted binary version!
